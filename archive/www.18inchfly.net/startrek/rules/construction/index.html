Ship Construction Manual MCA

This Rules Modifications, Clarifications, and Additions (MCA) provides updates to the Ship Construction Manual of the Starship Tactical Combat Simulator rules. It addresses some of the gaps and omissions in the original rules and adds rules to handle situations not otherwise covered.


Download MCA (V2.5, Word, 52K)

Other Ship Construction Manual Updates

The following files contain information on ship construction gathered from a number of STCS sources. Unless otherwise indicated, this information does apply to the local ship designs, and to ships used in the campaigns on this site. All page references apply to the Ship Construction Manual.

    • Federation New Technical Data (Zipped Excel document, 5K)
    These tables add new information to the following existing tables:
      Control Computer System Types (page 40)
      Warp Engine Types - Single Engine Use (page 41)
      Warp Engine Types - Tandem Engine Use (page 41)
      Shield Generator Types (page 46)
    • Federation Movement Ratio Tables (Zipped Excel document, 11K)
    These tables replace the Movement Point Ratio Table: Single Warp Engine and Movement Point Ratio Table: Tandem Warp Engines on pages 42 and 43. They include engines and entries not on the original tables.
    • Federation Monitor Movement Ratio Tables (Zipped Excel document, 5K)
    These tables provide movement point ratios for single and tandem monitor engines.
    • Federation Maximum Shield Power Table (Zipped Excel document, 6K)
    This table replaces the Maximum Shield Power table on page 46. It includes shield generators not on the original table.
    • Federation Other Design Information (Zipped Excel document, 5K)
    These tables provide design information for equipment not included in the original construction rules. Included are:
      Single Warp Engines For Monitors
      Tandem Warp Engines For Monitors
      Matter / Anti-Matter Engines For Space Stations
      Impulse Engines For Space Stations
      Space Station Costs
      Space Station Shields
    • Klingon Other Design Information (Zipped Excel document, 5K)
    These tables provide design information for equipment not included in the original construction rules. Included are:
      Single Warp Engines For Monitors
      Tandem Warp Engines For Monitors
      Space Station Costs
    • Construction Info Tables (Zipped Excel document, 4K)
    These are three commonly used tables which are spread out in the text of the manual instead of being back with the construction tables where they are needed. Included are:
      Weapon Ranges
      Ship Class Sizes
      Crew Casualties Table
