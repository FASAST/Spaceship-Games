Star Trek Campaigns
Game Forms

These are forms used to support tactical play of the STCS.

    • Turn Sequence Form (Version 3.4, zipped Excel document, 5K)
    This is a check-off form for the segments and phases in a turn.
    • Summary Charts (Version 3.0, GIF file, 31K)
    This is a page with charts and rules summaries to help speed play.
    • Control Panel, Ship (Version 2.5, zippped GIF files, 42K)
    This contains two graphics; the front page and the back page of the standard control panel for a ship.
    • Ship Control Panel, Station (Version 2.5, zippped GIF files, 44K)
    This contains two graphics; the front page and the back page of the standard control panel for a space station.
    • Ship Control Panel, Ground (Version 2.5, zippped GIF files, 43K)
    This contains two graphics; the front page and the back page of the standard control panel for a ground installation.
