Individual Counter Codes

This is a list of the codes used on the counters.

Column   Contents
 
Counter Code   The four character code which appears on a counter. The first three characters are listed here. A "#" is shown where the Individual Identifier would be on the counter.
Nation / Race   The nation to which the ship belongs (ex. Federation).
Type   The Type of a ship (ex. Battleship, Cruiser, etc.).
Class   The specific class of a ship (ex. Enterprise, D-7, etc.).
Sheet   The sheet on which the counter is located.
#   The number of counters on the sheet.
