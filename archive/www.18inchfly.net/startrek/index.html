Star Trek Campaigns

This site has been created to support play in Star Trek campaign games run and being run in Fort Wayne, Indiana. All campaigns are set in FASA's Star Trek Role Playing Game system, and specifically use a modified version of the Starship Tactical Combat Simulator (STCS) for tactical engagements.
