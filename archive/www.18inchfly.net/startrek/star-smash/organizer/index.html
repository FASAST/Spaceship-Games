Star Smash - Squadron Organizer

The Squadron Organizer is an Excel spreadsheet used to select ships for the squadrons in the tournament.

The current version of the Squadron Organizer is 1.8.

Using The Squadron Organizer

    Preparation
        Before starting to organize a squadron, you may want to have the Master Ship Listing available (either the Excel file or the on-line version). Use it as a reference for selecting ships. You may also want to have access to the design information (the Designs section of this site, and the STSSTC - Millennium Update & Archive).
    Select Nation
        Indicate the nationality of the squadron by placing an "x" (or almost anything) in one of the white boxes at the upper left of the spreadsheet.
    Select Ships
        Select each of the ships to be included in the squadron. Type the complete name of the ship or copy and paste the name from the other tabs in the Squadron Organizer file.
    Selection Errors
        If any Red information is displayed on the spreadsheet, you have an error and need to change your selection(s) to remove the illegal condition.
    Entry Errors
        If any of the computed data (light gray areas) display non-numeric values such as "N/A", "Value", or "##", then you have made an entry error. Check to see that you have selected a nation and that the ships entered are correct for the nation selected.
