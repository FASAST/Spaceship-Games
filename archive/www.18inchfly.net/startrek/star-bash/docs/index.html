Star Bash - Campaign Documents Needed

    Programs Needed:
     
    • Microsoft Word (or a program which can read Word documents)
    • Microsoft Excel (a version which is able to handle Workbook files)
      or a Microsoft Excel reader program for those who do not have Excel.
    • Any program which can display GIF and JPG files
      (an internet browser will do if you have nothing else)

    Documents Needed (Downloads):
     
    I have been told that some of the zipped Excel files cannot be unzipped. It appears that the MacZip program I got just for this purpose doesn't do its job correctly. If you cannot unzip a file let me know and I will e-mail the original file.
     
    • Star Bash Rules (Version 1.4, Word document)
    These are the rules used for campaign level play. All players need to have some knowledge of these rules, if only so they can write orders in a somewhat coherent manner.
     
    • Squadron Organizer (Version 2.2, Zipped Excel document)
    This Excel spreadsheet helps commanders to select the ships which will be part of their squadrons.
     
    • Movement Computer (Version SB1.0, Zipped Excel document)
    This Excel spreadsheet calculates the distance and travel time between systems and locations in the combat sector. Its values are the "official" ones for the campaign.
     

    Documents Needed (Reference):
     
    • Master Ship Listing
    The Master Ship Listing lists ships usable in the campaign.
     
    • Star Trek Rules MCA
    This is the Rules Modifications, Clarifications, And Additions (MCA) for FASA's Starship Tactical Combat Simulator. Along with the original FASA rules these are the tactical rules used for battles. This is an optional item for campaign players.
