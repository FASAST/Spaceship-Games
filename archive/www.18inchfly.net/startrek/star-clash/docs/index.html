Star Clash - Campaign Document Downloads

The following documents support the play of the campaign. All players should have copies of these documents (except those restricted by side).

    Programs you need:
        Microsoft Word (or a program which can read Word documents)
    Microsoft Excel (a version which is able to handle Workbook files)
      or a Microsoft Excel reader program.
    Any program which can display GIF and JPG files (an internet browser will do if you have nothing else)

Everyone:
 
• Campaign Rules (Version 2.6, Word document)
   These are the rules used for campaign level play. All players need to have some knowledge of these rules, if only so they can write orders in a somewhat coherent manner.
 
• Triangle Map (Version 1.0, JPG)
   This is a simple black and white map showing the locations of the systems in The Triangle. Only systems and locations shown on this map are included in the campaign.
 
• Day 000, Status Map (Version 1.0, JPG)
   This is a color version of The Triangle map showing who controls which systems and locations at the start of the war. It is a good initial reference, especially for those who need color to catch their attention. This map will be updated and sent to the players as the situation changes, so if you don't have it now, don't worry about it.
 
• Master Ship Listing (Version 1.3.4, Zipped Excel document)
   This is a listing of all the ships which may be found in use in the campaign. It includes "official" FASA ships and local designs. Note that it is not a complete listing of all possible ships; only those in service at 2/2500 and soon thereafter. Also note that the file has been updated since the beginning of the campaign. Check the Version number in your copy to see that you have the latest version (look in the Notes tab).
 
• Intelligence Briefing on The Triangle (Version 1.1, Word document)
   This document includes all the information commonly available regarding the situation in The Triangle at the start of the war. Most importantly, it describes the nations and systems in that section of space. It is clear from questions being asked that some of the players are not bothering to read this document before asking (and so are incurring numerous negative brownie points).
 
• Fleet Order Form, Blank (Version 1.1, Zipped Excel document)
   This is the form that everyone is supposed (hint ... hint) to be using to order their fleets from place to place.
 
• Fleet Listing, Blank (Version 1.1, Zipped Excel document)
   This is the form used by players to track the composition and status of their fleets. As the Campaign Master issues updated master fleet listing from time to time, it is unlikely that a player will ever need this form ... but it is here just in case.
 
• Movement Computer (Version 2.2, Zipped Excel document)
   This is an important file! It calculates the distance and travel time between systems and locations in The Triangle. Its values are the "official" ones for the campaign. Note that this file has been updated since the beginning of the campaign. If you have any Version before 2.1, throw it away and download this version! There was a serious error in Versions before 2.0.
 
• Star Trek Rules MCA (Version 7.0, Word document)
   This is the Rules Modifications, Clarifications, And Additions for FASA's Starship Tactical Combat Simulator. Along with the original FASA rules these are the tactical rules used for games. This is an optional item for campaign players. Some have requested a copy, so here it is.
 
Restricted Distribution:
   Only some of the players will receive each of the following documents. If you need copies, contact the Campaign Master via e-mail.
As the campaign is over, this information is available for anyone to download. The Fleet Listings are not included, as they are available for download from the Reports. The Fleet Location Listings are not included, as they are now in HTML format in the Reports.
