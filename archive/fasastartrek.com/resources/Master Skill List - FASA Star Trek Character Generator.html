Master Skill List

Pulled from FASA published materials and culled from web sources, this list (to the best of my ability) contains every skill that can be learned during character creation. I’m indebted to The Crimson Pirate.

 Note: Skills in bold do not receive a rating, a sub-skill must be chosen. Items preceded by { are sub-skills. Sub-skills with an asterisk need a racial specialty selected.

[This list is still a work in progress. I’ll be adding more sub-skills that will be needed in the generator.]
