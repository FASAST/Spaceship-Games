Transporters- Can be used in all sort of creative ways. For rescue operation a transporter or transporter's can be used to beam endangered personnel to the ship. You can set it on a wide beam and grab as many of the crew as possible. The people will then be stored in the transporter buffer until they are reintegrated. There is a big power cost to this though. 1 point of power for the transporter's and another one for the memory buffers. If the emergency transporters are used too you can add another point of power to the cost (most emergency transporters are designed to beam out only, not for beam up). This will allow a double capacity for the transporters holding ability when the memory buffer is charged.
